Projects & Descriptions

Task Assignment Web Application - Final project developed to contruct tasks and assign them to each member of our group. Each task has its own specific deadline, objective, and real-time tracking progress counter. Bootstrap and ReactJS were implemented to provide website outlining and interaction with the users. Testest with unit testing Mocha. Hosted by Github.

(Upcoming) Python Coffee Maker project - Using Django for the backend and ReactJS for the frontend, this project will be a web application that allows users to create their own coffee recipes and save them to their account. Users will be able to view other users' recipes and rate them. The project will also include a coffee maker simulator that will allow users to test their recipes and see how they turn out. The project will be hosted on Github.
